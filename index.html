API Notes

Chatterbot systems are generally not pretrained and use Wikipedia APIs to get the data they use. Internet is usually required to use the APIs and therefore run the models
If contributers of Wikipedia or Wikimedia and their repspective APIs see any potential problems reguarding copywrite or API guidelines, please add an issue to the project at the Chatterbot Github and care will be taken to resolve the issue, up to, if nessesary, complete removal of the project.
